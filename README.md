SimpleTodo sebuah project sederhana untuk belajar membuat aplikasi web dengan menggunakan framework nextJS dan menggunakan database supabase. 

## Permasalahan
Organisasi sering mengirimkan sebuah dokumen ataupun file yang harus di kerjakan oleh anggota organisasi. Namun seringkali anggota organisasi lupa akan tugas yang diberikan oleh organisasi. 
## Solusi
Membuat sebuah aplikasi web yang dapat digunakan untuk mengelola tugas yang diberikan oleh organisasi kepada anggota organisasi. 

## Fitur
    - Live Chat Group
    - Pengumpulan file dengan batas tenggat dan prioritas
    - Dashboard untuk melihat tugas yang diberikan
    - Role based access control (Admin, User)
